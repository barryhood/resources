PDS Object COnfigurator

Allows you to cojfigure the timestamps of fincalc entries on a PDS object to test different expiry scenarios.

Instructions:

  1. Copy your PDS object from localStorage, paste into the first panel and hit parse
  2. If there are any configurable fincalc entries, they will be shown in the second panel where the dates can be set
  3. Once the dates are set, hit generate object to create a script which can be run in your browser's dev console to add the PDS object
